JASP Statistical Software Guide

Free and User-Friendly Statistics

Author

Masaki EGUCHI, Ph.D.

Modified

July 6, 2025

Overview

JASP (Jeffreys’s Amazing Statistics Program) is a free, open-source statistics program with a user-friendly interface. It’s designed to be intuitive for users familiar with SPSS while incorporating modern statistical methods including Bayesian analyses.

Why JASP for This Course?

  • Free and open-source: No license fees
  • User-friendly: Drag-and-drop interface
  • Real-time results: See analyses update as you work
  • Publication-ready output: APA-formatted tables
  • Reproducible: Save complete analyses
  • Modern statistics: Includes effect sizes and Bayesian options

System Requirements

Minimum Requirements

  • Windows: Windows 7 or higher, 64-bit
  • macOS: macOS 10.12 (Sierra) or higher
  • Linux: Ubuntu 16.04 or higher
  • RAM: 4GB (8GB recommended)
  • Storage: 500MB free space

Installation

Windows

  1. Visit jasp-stats.org
  2. Click “Download” → Windows
  3. Run the installer (.exe file)
  4. Follow installation wizard
  5. Launch JASP from Start Menu

macOS

  1. Visit jasp-stats.org
  2. Click “Download” → macOS
  3. Open the .dmg file
  4. Drag JASP to Applications
  5. First launch: Right-click → Open (security)

Linux

  1. Visit jasp-stats.org
  2. Download AppImage or use flatpak
  3. Make AppImage executable
  4. Run from terminal or file manager

Getting Started

Interface Overview

Three Main Panels

  1. Data View: Spreadsheet for data
  2. Analysis Panel: Available analyses
  3. Results Panel: Output and visualizations

Ribbon Menu

  • File operations
  • Edit functions
  • Common analyses shortcuts

Loading Data

Supported Formats

  • CSV files (recommended)
  • Excel files (.xlsx)
  • SPSS files (.sav)
  • Copy/paste from clipboard

Import Steps

  1. File → Open → Computer
  2. Browse to file location
  3. Select file
  4. Verify variable types

Course-Specific Analyses

Day 4, Session 1: Descriptive Statistics

Frequency Analysis

  1. Descriptives → Descriptive Statistics
  2. Select variables
  3. Choose statistics:
    • Mean, median, mode
    • Standard deviation
    • Range, min, max
  4. Enable plots:
    • Distribution plots
    • Box plots

Example: Corpus Word Length

Variables: word_length
Statistics: mean, SD, histogram
Split by: text_type

Day 4, Session 2: Group Comparisons

Independent Samples T-Test

  1. T-Tests → Independent Samples T-Test
  2. Dependent Variable: frequency_score
  3. Grouping Variable: learner_level
  4. Additional Statistics:
    • Effect size (Cohen’s d)
    • Confidence intervals
    • Assumption checks

Chi-Square Test

  1. Frequencies → Contingency Tables
  2. Rows: error_type
  3. Columns: proficiency_level
  4. Statistics:
    • Chi-square test
    • Cramér’s V
    • Expected counts

Day 4, Session 3: Correlation and Regression

Correlation Analysis

  1. Regression → Correlation Matrix
  2. Select variables
  3. Options:
    • Pearson’s r
    • Spearman’s rho
    • Confidence intervals

Linear Regression

  1. Regression → Linear Regression
  2. Dependent: accuracy_score
  3. Predictors: study_hours, vocabulary_size
  4. Statistics:
    • R-squared
    • Coefficients
    • Assumption checks

Data Preparation Tips

Variable Types

  • Nominal: Categories (e.g., L1_language)
  • Ordinal: Ordered categories (e.g., proficiency_level)
  • Scale: Continuous (e.g., test_scores)

Data Cleaning

  1. Check for missing values
  2. Verify variable types
  3. Look for outliers
  4. Create new variables if needed

Common Analyses for Linguistics

Comparing Groups

  • T-tests: Two groups
  • ANOVA: Multiple groups
  • Chi-square: Categorical data

Relationships

  • Correlation: Linear relationships
  • Regression: Prediction models
  • Logistic regression: Binary outcomes

Descriptive Summaries

  • Frequencies: Count data
  • Descriptives: Continuous data
  • Explore: Detailed examination

Output and Reporting

Tables

  • Right-click → Copy
  • Paste into Word/Google Docs
  • APA format ready

Plots

  • Right-click → Save as
  • High resolution for publications
  • Customizable appearance

Complete Analysis

  • File → Export Results
  • HTML or PDF format
  • Includes all output

Troubleshooting

Common Issues

JASP won’t open - Check system requirements - Update operating system - Try compatibility mode (Windows)

Data import fails - Check file encoding (UTF-8) - Remove special characters - Save as CSV and retry

Analysis grayed out - Check variable types - Ensure data is selected - Verify analysis requirements

Slow performance - Close other applications - Reduce data size for practice - Update JASP version

Best Practices

Before Analysis

  1. Understand your data
    • Variable types
    • Sample sizes
    • Missing values
  2. Plan analyses
    • Research questions
    • Appropriate tests
    • Assumptions to check

During Analysis

  1. Start simple
    • Descriptive statistics first
    • Build complexity gradually
    • Check assumptions
  2. Document steps
    • Save JASP file (.jasp)
    • Note decisions made
    • Keep analysis log

After Analysis

  1. Interpret carefully
    • Effect sizes matter
    • Consider practical significance
    • Report limitations

  5. Interpret for linguistics

Quick Reference

Key Shortcuts

  • Ctrl+O: Open file
  • Ctrl+S: Save analysis
  • Ctrl+C/V: Copy/paste data
  • F5: Refresh analyses

Analysis Decision Tree

Comparing groups?
├─ Yes
│  ├─ 2 groups → T-test
│  └─ 3+ groups → ANOVA
└─ No
   ├─ Relationship? → Correlation/Regression
   └─ Categories? → Chi-square
