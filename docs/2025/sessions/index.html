Overview

This page provides access to all course sessions organized by day. Each session includes lecture materials, hands-on activities, and additional resources.

Navigation Tips
  • Click on any day to see all sessions
  • Each session page includes downloadable materials
  • Activities have step-by-step instructions
  • Check “Pre-class” sections for preparation requirements

Course Schedule at a Glance

Day-by-Day Overview

Day 1: Introduction and Foundations

Build a solid foundation in corpus linguistics concepts and basic tools.

Morning Sessions - Course introduction and overview - What is corpus linguistics? - Types of corpora

Afternoon Sessions - Hands-on with AntConc - Basic corpus searches - Frequency analysis

Key Skills: AntConc basics, concordance reading, frequency lists

Day 2: Advanced Corpus Analysis

Explore sophisticated analysis techniques for deeper linguistic insights.

Morning Sessions - Collocation analysis - Statistical measures - Keyword analysis

Afternoon Sessions - Regular expressions - N-grams and clusters - Tagged corpus work

Key Skills: Collocation statistics, keyword extraction, pattern searching

Day 3: Learner Language Analysis

Focus on analyzing non-native speaker language and learner corpora.

Morning Sessions - Learner corpus design - Available resources - Research applications

Afternoon Sessions - Error annotation - Interlanguage features - L1 influence detection

Key Skills: Error analysis, contrastive analysis, developmental patterns

Day 4: Statistical Analysis

Apply statistical methods to validate corpus findings.

Morning Sessions - Descriptive statistics - Distribution analysis - Effect sizes

Afternoon Sessions - Hypothesis testing - Chi-square tests - Data visualization

Key Skills: JASP usage, statistical testing, creating visualizations

Day 5: Advanced Topics and Integration

Synthesize learning and explore cutting-edge applications.

Morning Sessions - Formulaic language - Multi-word units - Pedagogical applications

Afternoon Sessions - Project presentations - Peer feedback - Future resources

Key Skills: Project design, presentation skills, independent research

Session Resources

Standard Session Materials

Each session typically includes:

  1. Slides - Main presentation materials (PDF format)
  2. Activities - Hands-on exercises with instructions
  3. Data Files - Corpora and datasets for practice
  4. Readings - Session-specific articles or chapters
  5. Links - Additional online resources

Technical Requirements by Day

Day Required Software Data Files
Day 1-3 AntConc Provided corpora
Day 4 JASP, Excel/Sheets Statistical datasets
Day 5 All previous tools Student choice

How to Use Session Materials

Best Practices
  1. Before Class
    • Review session objectives
    • Complete required readings
    • Download necessary files
    • Test software functionality
  2. During Class
    • Follow along with demonstrations
    • Complete activities in real-time
    • Ask questions as they arise
    • Take notes on modifications
  3. After Class
    • Review completed activities
    • Try additional practice exercises
    • Prepare assignments
    • Explore supplementary resources

Troubleshooting Common Issues

Software Problems

  • AntConc won’t open:
  • JASP crashes: Ensure you have latest version
  • Files won’t load: Verify file encoding (UTF-8)

Data Issues

  • Corpus too large: Use provided samples first
  • Character encoding: Convert to UTF-8
  • Missing files: Check course repository

Session Recordings

Recording Policy

Sessions may be recorded for: - Students who miss class due to illness - Review purposes - Future course development

Recordings are not a substitute for attendance!
